# React Image Grid Animator
## Getting Started with React Image Grid Animator
### How to install
#### `npm install react-image-grid-animator`
### Usage

    <ImageGrid
		images={images}
		visibleCount={10}
		interval={2000}
		animationItemcount={3}
		transitionDuration={200}
		randomized={true}
		isActive={true}
	/>
